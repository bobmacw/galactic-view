# Galactic View — Interactive Milky Way Visualizer

An interactive browser-based visualization of the Milky Way's structure and seasonal visibility from 42°N latitude (Newton, MA).

## Live Demo

👉 [View the visualization](https://yourusername.github.io/galactic-view/galactic-interactive.html)

## What It Shows

- The Milky Way's major spiral arms and galactic features in 3D
- Seasonal visibility of the galactic plane from 42°N
- The relationship between the galactic and celestial coordinate systems
- Rotatable, interactive 3D model built in vanilla HTML/CSS/JavaScript

## Files

| File | Description |
|------|-------------|
| `galactic-interactive.html` | Main visualization — self-contained single file |

## Usage

Open `galactic-interactive.html` directly in any modern browser, or visit the live GitHub Pages link above. No build step, no dependencies.

## Background

Built as part of a self-directed astronomy learning project exploring the Milky Way's geometry, spiral arm layout, and how the galactic center's visibility changes through the year as seen from mid-northern latitudes.
